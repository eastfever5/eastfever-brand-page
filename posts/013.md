# AI와 함께하는 게임 기획 (2) — 옵시디언에 Copilot 플러그인 붙이기

지난 편에서 구글 드라이브 동기화 폴더를 옵시디언 작업 보관소(Vault)로 설정해 Gemini와 연동하는 환경을 구축했습니다. 이번에는 옵시디언 내부에서 직접 AI와 대화할 수 있도록 **Copilot 플러그인**을 설치하겠습니다.

## 최종 구성 모습

문서 작업 창 우측에 AI 채팅창이 나란히 열립니다. 작성 중인 문서와 관련된 질문을 AI에게 즉시 할 수 있고, AI가 내 문서를 참조해 답변합니다.

## Copilot 플러그인 설치 과정

### 1단계. 옵시디언 설정 진입

화면 우측 하단의 작은 **톱니바퀴 아이콘** 클릭 → 설정 화면 진입

### 2단계. Copilot 플러그인 설치

설정 화면 좌측 메뉴 → **커뮤니티 플러그인** → **탐색** 클릭 → "Copilot" 검색 → 설치

### 3단계. API Key 설정

설치된 Copilot 섹션의 톱니바퀴 아이콘 클릭 → 설정 화면에서 두 가지를 입력합니다.

**API Key 발급 주소**: [https://aistudio.google.com/app/api-keys](https://aistudio.google.com/app/api-keys)

구글 로그인 상태라면 바로 진입되며, 이미 생성된 키가 있다면 그것을 사용해도 됩니다.

**채팅 모델 선택**:
- Google One Pro 구독자라도 AI 스튜디오 유료 기능은 별도 구매 필요
- 무료 버전인 `gemini-1.5-flash`로도 충분히 활용 가능

### 4단계. 첫 사용

설정 완료 후 채팅창에 첫 메시지를 보내봅니다. 게임 기획서를 작성하겠다고 하면 양식 템플릿과 유용한 팁을 즉시 제공합니다.

## 이 환경의 실제 활용 방법

| 도구 | 역할 |
|------|------|
| 옵시디언 Copilot | 간단한 질문, 현재 문서 구조 관련 논의 |
| Gemini Pro (웹) | 깊은 추론, 검토, 복잡한 분석 |

옵시디언에서 작성한 문서는 구글 드라이브에 실시간 동기화되므로, Gemini 웹 버전에서도 해당 문서 내용을 복붙 없이 바로 참조하게 할 수 있습니다.

## 마무리

이제 게임 기획 작업 환경이 완성됐습니다:

- **Obsidian**: 마크다운 기반 문서 작성 및 계층 구조 관리
- **Copilot 플러그인**: 로컬 문서 기준으로 직접 AI 상담
- **Google Drive**: 문서 실시간 동기화 및 Gemini 연동
- **Gemini Pro**: 심층 분석 및 검토

다음 편에서는 이 환경에서 실제로 게임 컨셉 기획서를 작성한 과정을 소개하겠습니다.
