# AI와 함께하는 게임 기획 (1) — 실시간 AI 협업 환경 구축하기

새로운 게임 아이디어가 떠올랐습니다. 이번 기획 작업부터는 AI를 적극 활용해 **"실시간 AI 협업 환경"**을 만들어 보기로 했습니다. 기획 문서를 작성하는 내내 AI의 도움을 받을 수 있는 환경을 구축하는 것이 목표입니다.

## 환경 구성 요소

```
구글 드라이브  ── 실시간 동기화 ──  Obsidian (로컬 작업)
       │
       └── Gemini AI가 문서 참조 가능
```

### Step 1. 구글 드라이브 설치 및 "파일 스트림" 설정

Gemini는 구글 드라이브에 올려둔 문서를 참조할 수 있습니다. 기획 문서를 드라이브에 올려두면 AI가 항상 최신 내용을 기반으로 답변해줍니다.

1. 구글 드라이브 데스크톱 앱 설치
2. 설정에서 **"파일 스트림" 모드** 활성화
3. 기획용 드라이브 폴더를 오른쪽 버튼 클릭 → **"오프라인에서 사용"** 설정

이렇게 하면 해당 폴더가 내 컴퓨터 로컬에 실시간 동기화됩니다.

맥북의 경우 동기화 경로는 보통 다음과 같습니다:
```
/Users/[계정명]/Library/CloudStorage/GoogleDrive-[이메일]/[폴더명]
```

### Step 2. 옵시디언 설치 및 설정

Obsidian은 마크다운 문서를 계층 구조로 관리하는 데 특화된 도구입니다. 개발자 커뮤니티에서 표준이 된 마크다운 문법을 지원하며, 무료입니다.

1. [obsidian.md](https://obsidian.md)에서 설치
2. "보관함(Vault) 열기" 선택
3. **경로를 위에서 동기화한 구글 드라이브 폴더로 설정**

이제 옵시디언에서 작성한 문서가 바로 구글 드라이브에 반영됩니다.

### Step 3. Gemini와 연동

Gemini 채팅창에서 구글 드라이브 접근 권한을 허용하고, 기획 문서의 경로를 알려주면 됩니다.

```
"구글 드라이브의 [폴더명] 폴더에 있는 md 파일들을 참고해서 대화해줘."
```

## 이 환경의 장점

- 기획 문서를 작성하면 AI가 즉시 해당 내용을 반영한 답변을 줄 수 있습니다.
- 마크다운 형식으로 문서를 통일해 AI와의 소통이 더 명확해집니다.
- 문서가 구글 드라이브에 자동 백업되므로 별도 저장 걱정이 없습니다.
