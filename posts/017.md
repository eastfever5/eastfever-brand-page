# 바이브 코딩 도전, 쉽고 빠르게 이렇게 시작해보세요

바이브 코딩을 하고 싶은데 막막하다면, 오늘부터 이 방법으로 시작해보세요. 코딩 경험이 없어도 됩니다.

## 4단계 입문 가이드

### 1단계. 개발 주제 선정

**핵심 조건 3가지**를 모두 만족하는 첫 번째 앱을 정합니다:

1. 웹 페이지 내에서 동작하는 웹앱
2. **단 하나의 HTML 파일**로 동작하는 앱
3. 서버 통신이 필요 없는 앱

어떤 것들이 가능할까요? 생각보다 많습니다:
- 브랜드/인물 소개 페이지
- 가계부, 모임 장부
- 간단한 메모 앱
- 타이머, 계산기

---

### 2단계. AI와 함께 기획 및 설계

만들고 싶은 앱을 **한 문장**으로 설명하고, 다음 3가지를 정합니다:

1. 앱에 꼭 있어야 하는 핵심 기능 3가지
2. AI에게 "index.html 하나로 구현 가능하냐"고 물어보기
3. AI가 가능하다고 하면 "나한테 궁금한 거 있으면 물어봐"라고 하고 대화 이어가기

AI와의 대화를 통해 필요한 사항들이 하나씩 정해집니다. "이제 모든 준비가 됐다"는 느낌이 올 때까지 계속 대화합니다.

---

### 3단계. index.html 파일 생성

준비가 됐다면 AI에게 이렇게 요청하세요:

> "코드를 텍스트로 주지 말고, HTML 파일로 다운로드할 수 있게 해줘."

파일을 받으면 잘 보관합니다.

---

### 4단계. index.html 파일 실행

파일을 더블클릭해서 열면 될 것 같지만, 그게 잘 안 되는 경우가 많습니다. 가급적 **로컬 웹 서버**를 통해 실행하는 것이 좋습니다.

AI에게 이렇게 물어보세요:

> "내 컴퓨터 환경에서 HTML 파일을 실행할 수 있는 로컬 웹 서버 띄우는 방법 알려줘. 나는 [윈도우/맥북]이야."

AI가 알려주는 방법을 따라 실행하면 됩니다.

---

## 이제부터는 이 사이클의 반복

```
실행 → 확인 → AI 대화(수정 요청) → 다시 실행
```

이 작업을 반복하면서 앱이 점점 완성됩니다.

## 예전에는 몇 달이 걸렸던 일이

이제 AI와 씨름하면 **수십 분에서 수 시간** 안에 원하는 웹앱의 초안을 만들 수 있습니다. 물론 오류가 나고, 원하는 대로 안 되는 경우도 많습니다. 그래도 포기하지 않고 계속 AI와 대화하면서 나아가다 보면 어느 순간 완성에 가까워집니다.

다음 편에서는 이 방법으로 앱을 만든 후, 더 쾌적한 개발 환경을 갖추는 방법을 소개하겠습니다.
