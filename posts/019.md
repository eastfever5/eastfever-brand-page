# 바이브 코딩, 어떤 도구를 써야 할까? 5가지 비교 정리

바이브 코딩에 관심은 있는데 뭘 써야 할지 모르겠다는 분들이 많습니다. AI 개발 도구들이 워낙 빠르게 쏟아지다 보니 선택하기가 어렵습니다.

20년 개발 경력으로 다양한 도구를 써본 입장에서, 초심자가 시작하기 좋은 도구들을 정리해봤습니다.

---

## (1) VSCode — 무료, 검증된 범용 편집기

새로 바이브 코딩을 시작하는데 유료 AI 구독이 부담스럽다면, 기존에 쓰던 무료 AI에게 코드를 생성시키고 그 결과물을 VSCode로 가져와 수정하는 방식으로도 충분히 시작할 수 있습니다.

**장점**: 무료, 강력한 플러그인 생태계, AI 개발용 확장 도구 다수  
**단점**: AI와의 직접 통합이 제한적, 코드를 직접 붙여 넣어야 하는 번거로움

---

## (2) Lovable — 비개발자 친화적, 채팅→프리뷰 즉시 확인

좌측 채팅창에 원하는 것을 설명하면 우측 프리뷰 영역에 즉시 결과가 나타납니다. 말 한마디로 웹앱 형태를 뚝딱 잡아주는 것이 장점입니다.

**장점**: 비개발자도 바로 시작 가능, 시각적 피드백 즉시 확인  
**단점**: 웹 서비스 제작만 가능, 무료 크레딧 소진 후 구독 필요

---

## (3) Antigravity — VSCode에 Google AI를 통합한 환경

VSCode에 구글이 직접 개발한 AI 에이전트가 탑재된 개발 환경입니다. AI에게 명령을 내리면 VSCode 편집기에 결과물이 바로 반영됩니다. 기존 VSCode처럼 AI 결과를 복붙할 필요가 없습니다.

**장점**: VSCode의 강점 + AI 자동 코드 적용, Gemini 연동  
**단점**: 설치 환경 구성이 다소 필요

> 현재 주력으로 사용 중인 도구입니다. Gemini Pro 구독과 함께 사용합니다.

---

## (4) Claude Code — 현업 개발자들의 선택지 1위

최근 업계에서 가장 많이 언급되는 도구입니다. 개발자든 비개발자든 바이브 코딩에 진심인 분들이 몰리고 있습니다. 문제 해결 능력이 다른 AI보다 뛰어나다는 평가를 받고 있습니다.

**장점**: 뛰어난 문제 해결 능력, 바이브 코딩 커뮤니티에서 가장 활발  
**단점**: 터미널(명령줄 환경) 사용이 필요해 비개발자에게는 진입장벽이 있음

---

## (5) Codex (ChatGPT) — ChatGPT AI 기반 개발 환경

ChatGPT의 개발 특화 도구입니다. 최근 모델 업데이트로 성능이 크게 향상되었다는 평이 있습니다.

**장점**: ChatGPT 생태계 연동, 최신 모델 성능 우수  
**단점**: 개발 친화적 UI라 비개발자에게 적응 시간 필요

---

## 어떤 도구를 선택해야 할까?

| 상황 | 추천 도구 |
|------|---------|
| 비개발자, 빠르게 시작하고 싶음 | **Lovable** |
| 기존 개발자, 익숙한 환경 원함 | **VSCode 또는 Antigravity** |
| 개발 능력이 있고 최고 성능 원함 | **Claude Code** |
| ChatGPT 구독자, 개발도 같이 | **Codex** |

---

어떤 도구를 선택하느냐보다 **지금 당장 시작하는 것**이 더 중요합니다. 도구는 쓰다 보면 자연스럽게 자신에게 맞는 것을 찾게 됩니다. 각자의 사정과 취향에 맞는 도구로, 즐겁고 보람찬 바이브 개발 되시길 바랍니다.
